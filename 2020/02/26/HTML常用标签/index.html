关于a标签

点击跳转百度搜索

属于`行内`元素

属性

属性 描述
href 规定链接的目标URL
target 规定在何处打开目标 URL。仅在 href 属性存在时使用。
download 指定下载链接
rel 规定当前文档与目标 URL 之间的关系。仅在 href 属性存在时使用。

常见用法

  • 跳转至外部的网页
  • 跳转至内部的某个锚点
  • 跳转到邮箱或者电话等

关于img标签

发送一个GET请求，并显示一张图片。属于`行内`元素

属性 值
src 规定显示图像的URL
width 规定图像的宽度
height 规定图像的高度
alt 规定图像的替代文本
crossoriginNew 设置图像的跨域属性

小知识

CSS 里，可替换元素（replaced element）的展现不是由CSS来控制的，这些元素是一类外观渲染独立于CSS的外部对象。 典型的可替换元素有 img、object、video和表单元素，如textarea、input.某些元素只在一些特殊情况下表现为可替换元素，例如 audio和canvas。 通过 CSS content 属性来插入的对象 被称作 匿名可替换元素（anonymous replaced elements）

关于table标签

tr 元素定义表格行，th 元素定义表头，td 元素定义表格单元。

拓展

关于文档流，指的是元素按照其在 HTML 中的位置顺序决定排布的过程,主要的形式是自上而下(块级元素)，一行接一行,每一行从左至右(行内元素)，我们在制作网页的初期，使用一些简单且常用的标签并不会出现太大的差错，课一旦使用浮动和定位标签，就是会出现各种各样的问题，但本质上，大多都属于文档流的问题。
